# Cyber-Scout: Health and Security Dashboard for Terminal

A lightweight, terminal-based security dashboard for macOS.

---

## 🚀 Features
* **🌐 Network Monitoring:** Instant connectivity check with status indicators.
* **💾 Disk Health:** Visual breakdown of storage usage and capacity.
* **🔋 Battery Status:** Real-time power tracking (Percentage & State).
* **⚠️ Security Deep Scan:** Scans the Downloads folder for executable `.sh` scripts and lists them by name.
* **🎨 ANSI UI:** Professional color-coded output (Green/Red/Bold).

## 🛠️ Installation & Usage
1. Clone the repository:
   `git clone https://github.com/mimen05/cyber-scout.git`
2. Run the script:
   `python3 scout.py`
